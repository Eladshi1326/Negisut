# אתר התדמית — נגישות (Showcase)

אתר תדמית (Vite + React, עברית/RTL) לפיצ'ר כפתור הנגישות. כולל את הווידג'ט החי בתוך הדף, כך שמבקרים יכולים לנסות אותו.

## הרצה מקומית (לבדיקה)
לחיצה כפולה על **`הפעל-אתר.bat`** — פותח VS Code, מתקין תלויות, ומריץ את האתר על `http://localhost:5173` (כרום נפתח לבד). דורש Node.js מותקן.

## פרסום אונליין (GitHub Pages)
לחיצה כפולה על **`העלאה-לגיט.bat`** — דוחף את הקוד לריפו `Negisut`. GitHub Actions בונה ומפרסם אוטומטית.
האתר יהיה זמין ב: `https://eladshi1326.github.io/Negisut/`

### הגדרה חד-פעמית (רק בפעם הראשונה)
בריפו `Negisut` ב-GitHub: **Settings → Pages → Build and deployment → Source: GitHub Actions**.
אחרי זה כל דחיפה מתפרסמת לבד.

## עריכת תוכן
- טקסטים ופיצ'רים: `src/App.jsx`
- עיצוב: `src/index.css`
- הווידג'ט החי נטען מ-`index.html` (השורה של jsDelivr) — אותו קוד שמותקן באתרים.

## הקישור לתחתית הווידג'ט
אחרי שהאתר באוויר, הכתובת `https://eladshi1326.github.io/Negisut/` היא זו שתשים כקישור בתחתית כפתור הנגישות (`brandLabel` / לינק).
